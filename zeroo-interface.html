Zeroo Interface

A criação de uma interface do zero foi um grande desafio. A alta complexidade do projeto e o volume de informações sem precedentes transformaram a relação entre o desenvolvimento e os stakeholders em um ambiente dinâmico, repleto de trocas constantes de ideias e soluções. Participei ativamente de todas as etapas do processo, mantendo uma comunicação contínua com desenvolvedores e Product Owners (POs) para garantir o alinhamento e a entrega de um produto eficiente e alinhado às expectativas.

Nota: a abordagem dessa página é propositalmente generalista a fim de assegurar o sigilo de informação sobre o produto a pedido do cliente.

Design System
As diversas abordagens do sistema para cada área trás um aspecto único para cada asset, mas mantém a identidade do sistema e promove uma solução simples para uma ideia complexa.
Foi uma honra imensa fazer parte de uma ideia tão revolucionária e mirabolante, que não apenas desafia os padrões atuais, mas também promete transformar radicalmente a área de administração de restaurantes. Participar de um projeto com tamanho potencial de impacto, que busca simplificar processos complexos e elevar a eficiência do setor, foi uma experiência verdadeiramente enriquecedora. Saber que essa iniciativa pode mudar para sempre a forma como os restaurantes são gerenciados é algo que me enche de orgulho e motivação. Foi um privilégio contribuir para algo que pode deixar um legado duradouro no mercado.
