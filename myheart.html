Myheart

O projeto foi desenvolvido como trabalho final para o CalArts. A proposta central é um aplicativo que auxilia responsáveis por crianças com complicações cardíacas. O aplicativo conta com duas interfaces distintas: a interface para Apple Watch foi pensada especialmente para o público infantil, com ilustrações lúdicas e uma construção visual agradável, utilizando cores suaves e amigáveis. Já a interface para iPhone é voltada para os responsáveis, apresentando informações detalhadas e funcionais para o acompanhamento da saúde da criança.

UI/UX

App Design
O desenvolvimento desse aplicativo surge como uma resposta sensível e inovadora às necessidades de famílias que lidam com complicações cardíacas infantis. A dualidade das interfaces reflete uma preocupação cuidadosa com os dois públicos envolvidos: as crianças, que precisam de uma experiência leve e acolhedora, e os responsáveis, que demandam informações claras e precisas para o monitoramento da saúde.
